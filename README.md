# Pankaj Medical and General Stores

Pharmacy E-Commerce Web Application.

## Monorepo Project Structure
- `/client` — React + Vite frontend
- `/server` — Node.js + Express backend

## Quick Start
To bootstrap the project and install all dependencies:
```bash
npm run bootstrap
```

To run both frontend and backend concurrently in development mode:
```bash
npm run dev
```

To run linting and code formatting checks:
```bash
npm run lint
npm run format
```
